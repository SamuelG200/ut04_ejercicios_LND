Formatos Multimedia HTML

Formatos de Audio

MP3 (MPEG-1 Audio Layer III):

  • El formato más extendido y compatible
  • Buena relación calidad/tamaño

WAV (Waveform Audio File Format):

  • Calidad de audio sin compresión (mayor fidelidad)
  • Archivos más pesados que MP3

OGG (Ogg Vorbis):

  • Formato libre y abierto
  • Mejor compresión que MP3 con similar calidad

Formatos de Vídeo

MP4 (MPEG-4 Part 14):

  • El formato más universal para web
  • Usa códec H.264 para vídeo y AAC para audio

WebM:

  • Formato abierto desarrollado por Google
  • Usa códec VP8 o VP9 para vídeo y Vorbis/Opus para audio

OGV (Ogg Theora):

  • Alternativa libre a MP4
  • Compatible con navegadores modernos
